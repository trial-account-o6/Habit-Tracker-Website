# Habit-Tracker-Website

Monorepo boilerplate for a Habit Tracker web app.

## Project structure

- `frontend/` — React + Vite + TypeScript
- `backend/` — Express + TypeScript

## Prerequisites

- Node.js 20+
- npm 10+

## Install

```bash
npm install
```

## Run in development

```bash
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:3001/health

## Build

```bash
npm run build
```
